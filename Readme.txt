# Bat-Rat Interaction Study

This repository contains data and analysis for a study on bat vigilance and risk-taking behavior in response to rat presence and seasonal changes.

## Repository Structure
- `data/` : Original and processed datasets
- `notebooks/` : Jupyter notebooks for preprocessing, analysis, and visualization
- `code/` : Python scripts for reusable functions
- `figures/` : Generated plots
- `report/` : Final report and tables

## How to Run
1. Install dependencies from `requirements.txt`
2. Open notebooks in `notebooks/` and run cells in order
3. Figures will be saved to `figures/` automatically
